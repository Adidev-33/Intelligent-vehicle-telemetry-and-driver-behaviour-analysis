ℹ About Intelligent Vehicle Telemetry System

Project Overview

This system uses real-time AI models to analyze vehicle telemetry data and classify driving behaviour as Smooth, Normal, Aggressive or Rash.

It integrates:

✔ RandomForest ML classifier ✔ LSTM time-series prediction ✔ Explainable AI ✔ Real-time OBD-II data streaming ✔ Dynamic RPM detection

Designed for competition-level automotive AI systems and optimized for Raspberry Pi deployment.
